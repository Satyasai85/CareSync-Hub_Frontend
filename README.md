# CareSync Hub Frontend

ReactJS frontend for the Hospital Appointment & Patient Management System.

## Setup

```bash
npm install
copy .env.example .env
npm run dev
```

Default app URL: `http://localhost:5173`

The backend must be running at `VITE_API_URL`, defaulting to `http://localhost:5000`.

## Screens

- Hospital analytics dashboard
- Doctor listing page
- Appointment booking page
- Patient dashboard
- Doctor dashboard
- Medical history page

## Demo Access

Use the role selector in the header to simulate `admin`, `receptionist`, `doctor`, and `patient` API permissions. Seed demo accounts are documented in the backend README.
